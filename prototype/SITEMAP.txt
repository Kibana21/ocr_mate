Navigation Flow:
================

Primary User Journey:
─────────────────────
index.html
    ↓ Click "Create Document Type"
02-schema-editor.html
    ↓ Click "Next: Add Ground Truth"
03-annotation.html
    ↓ After labeling, click "Optimize"
04-optimization.html
    ↓ After optimization completes
05-results.html
    ↓ Process more docs or return
index.html

Alternative Flows:
──────────────────
• Settings → Model Selector → Re-optimize
• Dashboard → Process → Results
• Any page → Top nav → Any other page

Interactive Features:
=====================

Highlighting:
• Hover on extraction field → highlights source in document
• Click highlighted text in document → fills field

Buttons & Actions:
• All buttons have hover states
• Primary actions are blue
• Danger actions are red
• Success actions are green

Form Elements:
• Focus states with blue outline
• Validation on required fields
• Auto-fill from OCR
• Click-to-copy functionality

Progress:
• Animated progress bars
• Real-time updates (simulated)
• Step-by-step indicators
